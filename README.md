# shahad-Mechanical-Engineering-Task
## Overview
This project was completed as part of the Mechanical Engineering track. The objective was to recreate a reference image as a 3D CAD model using Onshape, then export it as an STL file for manufacturing or 3D printing.

## Project Details
- Selected a Spider-Man circular logo as the reference image.
- Designed the model in Onshape.
- Created the circular profile and applied the required dimensions.
- Exported the final design as an STL file.
